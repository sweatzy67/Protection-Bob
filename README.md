# 🚀 Installation

```bash
git clone https://github.com/yourusername/Protection-Bob.git

cd Protection-Bob

pip install -r requirements.txt

python app.py
```

---

# ⚙️ Configuration

1. Create a Discord Bot.
2. Enable all required Gateway Intents.
3. Copy your Bot Token.
4. Create a `.env` file.
5. Paste your token:

```env
TOKEN=YOUR_BOT_TOKEN
```
